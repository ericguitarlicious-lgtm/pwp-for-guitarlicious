## 5. Pro Tips for GitHub Portfolios

1.  **Uppercase Functions:** GitHub highlights `=SUM()` better than `=sum()`.
2.  **No Spaces:** Ensure there is no space between the backticks and the language name (e.g., ` ```excel `, not ` ``` excel `).
3.  **The "Copy" Button:** GitHub automatically adds a "Copy" button to any block created with triple backticks.
4.  **Preview Shortcut:** In VS Code, press **Ctrl + Shift + V** to see this cheatsheet in action!
